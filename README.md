# unimporant-key-box-cli

###### A command line application is used to record passwords that are not important.

###### 一个命令行应用用于记录保存不重要的密码。

### usage

+ requires

  + nodeJS
  + npm

+ install

  ```shell
  npm install -g unimportant-key-box-cli
  ```

+ use

  ```shell
  KeyBox
  
  Keybox -h
  ```
